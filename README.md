# DEKRIS React + Vite Full Mock

## İşə salmaq

```bash
npm install --no-audit --no-fund
npm run dev
```

Aç: http://localhost:5173/login

## Funksiyalar

- Login / logout / protected routes
- Burger menu: desktop collapse, mobile sidebar açılıb-bağlanır
- Dashboard modul kartları
- Sənədlərin elektron dövriyyəsi: search, status tabs, yeni müraciət, status dəyişmə, imzala, yoxla, təsdiq et, imtina, sənəd listi, qeyd əlavə etmə
- Hüquqların qeydiyyatı səhifəsi
- Dövlət reyestri / əmlak kartı: search, tabs, hüquqlar, sənədlər, tarixçə, kadastr xəritəsi mock, PDF export/print mock
- Dövlət kadastrı səhifəsi
- Çoxmənzilli yaşayış binaları: MTK, pay, üzv, qərar məlumatları
- Sorğu: müraciət və əmlak üzrə vahid axtarış, çatışmayan sənəd tələbi
- Təhlil və hesabat: KPI, SVG line chart, bar chart, donut chart, region aktivliyi, export mock
- Məlumatların verilməsi: çıxarış, texniki pasport, kadastr planı və arayış yaratma mock
- Parametrlər: profil update, bildirişlər, şifrə mock, demo data reset
- LocalStorage ilə məlumatlar qalır
